# 🚀 ERP Management System

A Full Stack Enterprise Resource Planning (ERP) Management System built using:

- React.js
- Spring Boot
- PostgreSQL
- Bootstrap / CSS

---

# 📌 Features

## 👨‍💼 Employee Management
- Add Employees
- Update Employees
- Delete Employees
- Employee Search

## 🏢 Department Management
- Add Departments
- Manage Department Details

## 📅 Attendance Management
- Mark Attendance
- Attendance Records

## 💰 Salary Management
- Add Salary
- Salary Tracking
- Total Salary Calculation

## 📝 Leave Management
- Apply Leave
- Leave Status Management

## 📦 Inventory Management
- Add Inventory
- Manage Product Quantity
- Supplier Details

## 📊 Analytics Dashboard
- Employee Analytics
- Salary Charts
- Department Salary Distribution

## 📈 Finance Management
- Revenue Tracking
- Expense Tracking
- Profit Calculation

## 🎯 Recruitment Management
- Add Candidates
- Application Status Tracking

## ✅ Task Management
- Assign Tasks
- Task Priority
- Task Status

## ⭐ Performance Management
- Employee Reviews
- Ratings Management

## 👤 Profile Management
- Admin Profile
- Update User Information
